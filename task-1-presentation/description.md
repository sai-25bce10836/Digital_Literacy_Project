
Brief description

Students must produce a single-page visual infographic or slide which demonstrates their understanding of Digital Literacy and its value for student learning. The design requires Canva and Prezi and Miro tools to create simple yet captivating and user-friendly visual presentations. The visual must cover at least three key areas, such as:

What digital literacy means (ability to use digital tools effectively and responsibly)

Students need to learn about three digital tools which include Google Docs and GitHub and LinkedIn

Safe internet practices (privacy, strong passwords, avoiding scams)

Professional online presence (maintaining a clean and credible profile)

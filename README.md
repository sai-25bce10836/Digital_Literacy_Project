PROJECT OVERVIEW - The project requires students to build a Digital Literacy Portfolio which will help them acquire basic digital competencies. The program requires participants to complete five specific tasks which include creating an infographic and establishing an online professional presence and learning about coding and teamwork platforms and mastering email communication standards and cybercrime detection skills. The objective requires people to use digital technology properly while assisting others in achieving digital safety and efficiency through their online activities.

MODULE SUMMARIES

Module 1: Created a digital literacy infographic to spread awareness about safe and effective use of technology.

Module 2: Built a professional digital portfolio using platforms like GitHub and LinkedIn.

Module 3: Practiced coding on a platform and created a Google Form for collaboration and data collection.

Module 4: Learned professional email writing and responsible social media communication.

Module 5: Studied cybercrime types and created awareness through a case study and safety checklist.
